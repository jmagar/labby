Run the broadcastr CLI: `${CLAUDE_PLUGIN_ROOT}/bin/broadcastr $ARGUMENTS`.

Examples:
- `/broadcastr status`
- `/broadcastr recent --since=10m`
- `/broadcastr mute plan-exec`
- `/broadcastr emit cli info "manual broadcast"`

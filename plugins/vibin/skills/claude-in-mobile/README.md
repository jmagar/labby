# Claude in Mobile

Use when controlling or testing Android devices, iOS Simulators, macOS desktop apps, Aurora OS devices, Chrome/Chromium browser sessions, or app-store releases through the claude-in-mobile MCP server or native CLI.

## Usage

Invoke this skill when the user request matches the trigger conditions in `SKILL.md`. The skill body is the source of truth for workflow steps and operational constraints.

## Files

- `SKILL.md` - agent workflow and trigger guidance
- `agents/` - OpenAI runtime metadata
- `references/tooling.md` - setup commands, platform notes, and coordinate handling details
- `README.md` - packaging overview
- `CHANGELOG.md` - packaging change history

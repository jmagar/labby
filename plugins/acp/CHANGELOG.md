# Changelog

## [1.0.0] — 2026-06

Initial release of the ACP plugin.

- Rust ACP skill covering the full session lifecycle, streaming notifications, tool call wiring, and `?Send` + `LocalSet` runtime requirements.
- Reference files for wire format, message types, tool calls, Codex production patterns, and unstable features (reviewed against agent-client-protocol 0.13.1).
- Example agent and client implementation skeletons.
